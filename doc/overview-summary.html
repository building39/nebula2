Nebula is a Cloud Data Server based on the CDMI Version 1.1 specification, published by the Storage Network Industry Association (SNIA).

More information on CDMI can be found here. A PDF version of the specification can be downloaded from here.

Server

The current implementation is very basic, allowing for the creation of domains, domain members, realm maps, containers, data objects, and capability objects. HTTP basic authentication is supported, but ACLs are not yet supported. ACL support is a high priority task on the current roadmap.

Clients

Currently, Nebula provides only the server code - no client code. Various clients (web-based and mobile) are on the roadmap.
